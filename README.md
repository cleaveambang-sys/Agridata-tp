# 🌾 AgriData Cameroun

Application web de collecte et d'analyse descriptive des données agricoles,
développée dans le cadre du cours **INF 232 EC2 — Analyse de données**.

## Fonctionnalités

- Saisie des rendements agricoles et données météorologiques
- Analyse descriptive complète (moyenne, médiane, écart-type, corrélations)
- Visualisations interactives (histogrammes, box plots, scatter plots, heatmaps)
- Import / Export CSV
- Base de données SQLite intégrée

## Technologies utilisées

- Python
- Streamlit
- Pandas
- Plotly
- SciPy
- SQLite

## Lancer l'application localement

pip install -r requirements.txt
streamlit run agridata_app.py
